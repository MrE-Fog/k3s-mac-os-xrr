# k3s-mac-os-x

Simple script to install local kubernetes cluster on mac os x via multipass and ubuntu via local run

## Usage for mac os

Just run macos.sh:

```sh
./macos.sh
```

## Usage for ubuntu

Just run ubuntu.sh with sudo:

```sh
sudo ./ubuntu.sh
```
